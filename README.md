# Boxing
Open AI GymのBoxingにおいて、AIに遅延を与えることで難易度調整ができるか検証する

## 使用環境

- python=3.10.*
- tensorflow=2.10.0

## Gymnasiumのインストール
```
pip install gymnasium
pip install gymnasium[atari]
pip install gymnasium[accept-lom-ricense]
```

## その他のインストール
```
pip install keras-rl2
pip install tensorflow==2.10.0
```

## 学習方法
```
python Boxing-rl.py
```
