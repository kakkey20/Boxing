# Boxing
Open AI GymのBoxingにおいて、AIに遅延を与えることで難易度調整ができるか検証する

## 使用環境

- python=3.10.*

## Gymnasiumのインストール
```
pip install gymnasium[atari]
pip install gymnasium[accept-lom-ricense]
```

## 学習方法
```
python Boxing-rl.py
```
